# 福祉施設紹介スライダー用 写真

home.html「福祉の牧場として」セクションの自動スライダーで使用する 5施設の写真。

## ファイル名(必須)

| ファイル名 | 内容 |
|---|---|
| `kodomo-support.jpg` | こども家庭サポートセンター |
| `diakonia.jpg` | ディアコニア(特別養護老人ホーム) |
| `kohituji.jpg` | こひつじ(診療所) |
| `ibuki.jpg` | いぶき(就労支援) |
| `makibanoie.jpg` | まきばの家(児童養護施設) |

## 推奨スペック

- 比率: **4:3 推奨**(16:9 でも可、自動 cover トリミング)
- 幅: 800px〜1600px
- 形式: JPG(PNG/WebP も可、ファイル名拡張子をHTML側と合わせて変更)
- ファイルサイズ: 各 200KB 程度を目安(GitHub PagesのCDN配信)

## デプロイ

ファイルをこのフォルダに置いて `git push` するだけで自動反映。

参照URL: `https://oki-filmpresso.github.io/denmark-bokujou-pages/iframe/photos/welfare/<filename>.jpg`
